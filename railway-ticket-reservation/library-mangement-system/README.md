# Library Management System

The Library Management System is a console-based C application created for academic purposes. 
It helps manage basic library operations in a simple and organized manner.

## Features
- Add new book records
- Display available books
- Search books by title or author
- Issue books to students
- Return issued books
- Maintain records using file handling

## Concepts Used
- Structures
- File handling
- Arrays
- Menu-driven programming

## Language
- C
