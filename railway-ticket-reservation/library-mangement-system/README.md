# Student Exam Record System

The Student Exam Record System is a console-based C application developed as part of academic coursework. 
It is designed to manage student examination records efficiently using structured programming concepts.

## Features
- Add and store student details such as roll number, name, and branch
- Enter marks for multiple subjects
- Calculate total marks, percentage, and grade
- Display individual student results using roll number
- Display all student records in a structured format
- Store and retrieve records using file handling

## Concepts Used
- Structures
- File handling
- Conditional statements and loops
- Modular programming

## Language
- C
